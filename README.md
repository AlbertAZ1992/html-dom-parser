# html-dom-parser

An HTML to DOM parser that works on both the server and client.

## Installation

```sh
$ npm html-dom-parser
```

## Testing

```sh
$ git clone https://github.com/remarkablemark/html-dom-parser.git
$ npm install
$ npm run lint
$ npm test
```
